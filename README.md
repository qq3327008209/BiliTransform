## ✨ 核心功能

- 🔄 **视频旋转 (0°–360°)：** 滑条微调 / 预设按钮(0°,90°,180°,270°)
- 🔍 **视频缩放 (50%–400%)：** 自由缩放，预设 100% / 170% / 240% / 320%
- 🖱️ **中键滚动调节：** 鼠标悬停在滑条上，滚动滚轮精准微调
- 👆 **长按拖动：** 按住左键 500ms 后移动拖动视频位置（不影响播放/暂停）
- 🎯 **一键复位：** “重置视频位置”按钮，瞬间回到中心
- 🌈 **优雅动效：** 图标动画、滑条预览、毛玻璃面板

## 🎮 交互详解

### 🔹 旋转 & 缩放面板

- 鼠标悬停在播放器控制栏的 **旋转图标** 上，弹出控制面板。
- 拖动滑条或点击下方预设按钮，实时生效。
- 滑条支持 <kbd>⬆️</kbd> <kbd>⬇️</kbd> 滚轮调节（步长1% / 1°）。

### 🔹 拖动视频位置（智能长按）

- **长按 500ms 后移动鼠标** 才激活拖动模式 —— 避免短按误触播放/暂停。
- 激活后鼠标变为抓握手势，可随意移动视频在画面中的位置。
- 拖动期间会临时拦截点击事件，松开鼠标后恢复。
- 若想重置位置，点击面板底部的 **“重置视频位置”** 按钮。

> ⚠️ **注意：**
> 视频旋转/缩放会改变坐标系，拖动方向会适配旋转后的视角（即在旋转后的坐标系中拖动）。
